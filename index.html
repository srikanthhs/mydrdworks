Apps Script — How It Works
Architecture:
Google Sheet (source) → Apps Script (API layer) → This Web App (display + analytics)

What the script does:
• Reads all rows from the active sheet
• Returns JSON with headers + rows
• Supports optional ?block=…&stage=… query params
• CORS headers enabled for browser access

Google Drive integration:
• Can also read from a specific Sheet by file ID
• Supports multiple sheets/tabs
• Auto-detects date columns and formats them

Security:
• Deploy with "Execute as: Me" and "Access: Anyone"
• No login required for district users
• Use URL as the access key (keep it private)
Column Configuration
Change these in the Apps Script to match your sheet:
const CONFIG = { SHEET_NAME: "Sheet1", // tab name HEADER_ROW: 2, // row with column headers DATA_START: 3, // first data row DATE_FORMAT: "dd/MM/yyyy" };
